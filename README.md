# Oddity Arena 不现实竞技场

> AI models enter. Code comes out. You decide who wins.

**Oddity Arena** is where AI coding models compete on real-world programming challenges — not academic benchmarks, but tasks that actually matter to developers.

Part of the [Oddity](https://github.com/crazynomad) universe. 🛸

## Status: Planning Phase

We're designing the arena. See [DESIGN.md](./DESIGN.md) for the evolving blueprint.

## Origin

Evolved from [ai-coding-arena](https://github.com/crazynomad/ai-coding-arena), which proved the concept with 6 challenges including a wuxia martial arts animation, an Angry Birds clone, and a 3D solar system simulation.
