# Challenge 04: 鹈鹕骑自行车

生成一个鹈鹕骑自行车的 SVG 矢量图 — AI 界最著名的非官方基准测试。

## 起源

2024 年底，Django 共同创始人 Simon Willison 提出这个 Prompt 作为 AI 能力测试：鹈鹕和自行车在互联网数据中极少成对出现，能有效测试 AI 是否真正理解物体间的空间逻辑。

2026 年初随着 Gemini 3 发布，这个测试从开发者圈子出圈，成为 "AI 进化" 的文化符号。

## Prompt

Generate an SVG of a pelican riding a bicycle.

Requirements:
- Pure SVG output, viewable in any browser
- The pelican should be clearly recognizable (large beak/pouch, bird body)
- The bicycle should have two wheels, frame, handlebars, pedals
- The pelican should be ON the bicycle (seated, feet on pedals)
- Spatial logic must be correct (pelican above bike, wheels on ground)
- Style can be anything: realistic, cartoon, minimalist, artistic
- Must be a single HTML file with inline SVG
- No external dependencies

## 评测重点

- 🦆 鹈鹕辨识度 — 能一眼看出是鹈鹕吗？（大嘴、喉囊）
- 🚲 自行车结构 — 车轮、车架、把手、踏板是否完整？
- 🧠 空间逻辑 — 鹈鹕真的骑在车上吗？还是漂浮在旁边？
- 🎨 艺术表现力 — 风格、色彩、创意
- 😂 趣味性 — 有没有让人会心一笑的细节
